# Lessons go to the team catalog, not only personal memory
**When this applies:** the user says "take lesson", "save this learning", "document this for the team", or equivalent.
**Principle:** team-shared knowledge belongs in git (`.docs/`); personal memory only teaches future-me.
**Why:** on a team, a memory-only save means only the next session benefits, never teammates.
**How to apply:**
- Default to `.docs/common-issues/` (rules) or `.docs/principles/` (principles) first.
- Follow the Unified Save Protocol: scan, then append a dated sub-note rather than a sibling.
- Reach for private memory only when the learning is strictly personal-scope (profile, workflow quirk).
